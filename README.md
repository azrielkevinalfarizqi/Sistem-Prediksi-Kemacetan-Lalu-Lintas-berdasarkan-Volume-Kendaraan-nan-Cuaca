# 🚦Sistem prediksi kemacetan lalu lintas berbasis Machine Learning menggunakan data volume kendaraan dan kondisi cuaca.

📌 Deskripsi

Proyek ini membandingkan performa tiga algoritma regresi:

Random Forest Regression
Support Vector Regression (SVR)
K-Nearest Neighbors (KNN)

untuk memprediksi volume lalu lintas berdasarkan data historis dan faktor cuaca.

📊 Dataset

Dataset: Metro Interstate Traffic Volume Dataset

48.204 data
9 fitur
Sumber: UCI Machine Learning Repository
⚙️ Tahapan
Data Exploration
Data Preprocessing
Handling Noise
Handling Outlier
Feature Scaling
Model Training
Model Evaluation
Prediction & Visualization
🧪 Algoritma
Random Forest
SVR
KNN
📈 Evaluasi

Metode evaluasi yang digunakan:

MSE
RMSE
MAE
R² Score
🏆 Hasil Terbaik

Model terbaik diperoleh menggunakan Random Forest Regression dengan:

Feature Scaling
Feature Selection
Noise Handling (Median)

Model ini menghasilkan performa prediksi paling baik dibandingkan SVR dan KNN.
